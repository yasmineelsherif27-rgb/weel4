

Matrices – Linear Algebra
Matrix Definition

A Matrix is a rectangular arrangement of numbers organized in rows and columns.

Example:

A = | a11  a12  a13 |
    | a21  a22  a23 |

Order of matrix = rows × columns

Example:
2 × 3 matrix → 2 rows and 3 columns.

Types of Matrices
Row Matrix

Matrix that has one row only.

Example

[ 2  5  7 ]

Order → 1 × n

Column Matrix

Matrix that has one column only

Example

| 3 |
| 5 |
| 7 |

Order → n × 1

Square Matrix

A matrix where rows = columns

Example

| 1 2 |
| 3 4 |

Order → n × n

Transpose of Matrix

Transpose means interchanging rows and columns

Example

A = | 1 2 |
    | 3 4 |

Aᵀ = | 1 3 |
      | 2 4 |
Diagonal Elements

Diagonal elements are the elements from top left to bottom right

Example

| 1 2 3 |
| 4 5 6 |
| 7 8 9 |

Diagonal = 1 , 5 , 9

Trace of Matrix

Trace = Sum of diagonal elements

Example

Trace = 1 + 5 + 9 = 15
Zero Matrix

A matrix where all elements are zero

Example

|0 0|
|0 0|
Diagonal Matrix

Matrix where all elements are zero except diagonal

Example

|5 0 0|
|0 3 0|
|0 0 2|
Identity Matrix

Diagonal matrix where diagonal elements = 1

Example

|1 0 0|
|0 1 0|
|0 0 1|
Triangular Matrix
Upper Triangular Matrix

Elements below diagonal = 0

Example

|2 4 6|
|0 3 5|
|0 0 1|
Lower Triangular Matrix

Elements above diagonal = 0

Example

|2 0 0|
|4 3 0|
|6 5 1|
Matrix Properties
Addition
A + B = element by element addition

Example

A = |1 2|
    |3 4|

B = |2 3|
    |4 5|

A + B = |3 5|
        |7 9|
Scalar Multiplication
k(A + B) = kA + kB
Associative Property
A + (B + C) = (A + B) + C
Identity Property
A + 0 = A
