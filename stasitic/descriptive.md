Descriptive Statistics
Definition

Descriptive statistics is a branch of statistics that focuses on summarizing and describing data.
It helps us understand the dataset by using numerical measures and visual representations.

The main goal is to organize data and present it in a simple and meaningful way.

Types of Data

Data is generally classified into two main types.

1. Qualitative Data

Qualitative data describes categories or qualities and cannot be measured numerically.

Examples:

Gender

Color

Nationality

2. Quantitative Data

Quantitative data represents numerical values that can be measured.

Examples:

Age

Height

Weight

Salary

Quantitative data can be divided into:

Discrete Data

Data that takes countable values.

Example:
Number of students.

Continuous Data

Data that can take any value within a range.

Example:
Height or weight.

Measures of Central Tendency

Measures of central tendency describe the center of the dataset.

Mean (Average)

The mean is the average value of the dataset.

Formula:

Mean = Sum of values / Number of values

Example:

Data = 2, 4, 6, 8

Mean = (2 + 4 + 6 + 8) / 4 = 5

Median

The median is the middle value after sorting the data in ascending order.

Example:

Data = 1, 3, 5, 7, 9
Median = 5

If the number of values is even:

Data = 2, 4, 6, 8

Median = (4 + 6) / 2 = 5

Mode

The mode is the value that appears most frequently in the dataset.

Example:

Data = 2, 2, 3, 4, 5

Mode = 2

Measures of Dispersion

Measures of dispersion describe how spread out the data is.

Range

Range represents the difference between the highest and lowest values.

Range = Max − Min

Example:

Data = 2, 4, 7, 10

Range = 10 − 2 = 8

Variance

Variance measures how far each value in the dataset is from the mean.

A larger variance means the data points are more spread out.

Standard Deviation

Standard deviation is the square root of variance.

It measures how much the data deviates from the average value.

Small standard deviation → data values are close to the mean.
Large standard deviation → data values are widely spread.

Importance of Descriptive Statistics

Descriptive statistics helps in:

Organizing large datasets

Understanding patterns in data

Summarizing information

Preparing data for machine learning analysis
