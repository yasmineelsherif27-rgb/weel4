

Probability

Probability is a branch of mathematics that concerns the analysis of random phenomena and measures the likelihood of an event occurring.

Probability is expressed as a numerical value between 0 and 1, or as a percentage between them.

0 → the event is impossible to occur.

1 → the event is certain to occur.

The closer the value is to 1, the higher the chance that the event will occur.

Sample space:

S = {1,2,3,4,5,6}

Probability of getting 5:

P(5) = 1 / 6 ≈ 16.67%

Probability of getting 5 or 2 or 1:

P(5,2,1) = 3 / 6 = 50%
Related Probability Concepts
Conditional Probability

Conditional probability is the probability of an event A occurring given that another event B has already occurred.

It is denoted as:

P(A | B)

This is read as "the probability of A given B."

Example

A box contains:

8 red balls

6 blue balls

Two balls are drawn.

Probability that the first ball is red:

P(First Red) = 8 / 14

Probability that the second ball is red given the first is red:

P(Second Red | First Red) = 7 / 13
